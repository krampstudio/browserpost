Firefox 51

February 2017

New stable releases

Following the usual 6 weeks cycle, the 51st version of the Mozilla Foundation web browser becomes the stable version the 24th of January. As usual a lot of changes for developers. Among them, some noteworthy :

  • :indeterminate 65% #css
    :indeterminate is implemented for <input type="radio">
  • :placeholder-shown is implemented for <input type="text">
  • Symbol.toStringTag is implemented and is used to set the default description of an object [object Object]
  • const and let are now 100% ES2015 compliant, and you can now use const in for ... of loops.
  • WebGL2 is now enabled by default. Have a look to the brand new API based on OpenGL ES 3.0.
  • IndexedDB version 2 is now fully implemented
  • SVG nodes support new attributes : tabindex, href instead of xlink:href and the data attributes via datasets (data-*) #svg

Please consult the complete list of changes for web developers
